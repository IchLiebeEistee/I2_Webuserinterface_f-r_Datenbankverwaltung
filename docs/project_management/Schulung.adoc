== Theorie: Grundfunktionalitäten der Website-Nutzung

=== Einführung in die Benutzeroberfläche und Navigation
* Vorstellung der verschiedenen Elemente auf der Website (Menüs, Links, Suchfelder, etc.)
** Partnerhochschule hinzufügen + Vertrag anlegen + Studiengang anlegen + Mentor hinzufügen (Berichte)

* Hervorhebung der Hauptaspekte und deren Zweck
** Einzelne Reiter durchgehen und dort Anwendungen erläutern
* Erklärung spezifischer Funktionen wie Filter oder Suchfunktionen + versteckte Funktionen
